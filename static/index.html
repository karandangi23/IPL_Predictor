ℹ️ About the Model

🌲

Random Forest

An ensemble of 200 decision trees trained on 1,099 IPL matches. Each tree votes on the winner; the majority wins.

📦

14 Features

Team identity, venue, toss outcome, historical win rates, head-to-head record, average scores, and player fitness flags.

🎯

71% Accuracy

Evaluated on a held-out 20% test split (220 matches). Outperforms a naive baseline of ~55% from toss/win-rate alone.

🛠️

Tech Stack

Python · scikit-learn · Flask · HTML/CSS/JS. Fully local – no external AI API, pure ML inference.
